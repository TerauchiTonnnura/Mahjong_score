# Mahjong_score
Djangoを用いて麻雀のスコア管理をしよう．  
また，これを通してGitでのバージョン管理方法に触れよう．  


## 基本構想  

### 点数入力画面 (saki)  

#### 画面 1 
東風，半荘の選択を行う．  
各プレイヤの最終点数を入力する

#### 画面 2  
入力された点数が pt に変換されて，これまでの対局の各 pt と累積の pt を表示される．  
何かグラフとか出力しても良いかも．

### 管理者画面 (admin)  
新しいプレイヤの追加やウマオカの設定などを行う．  
詳しいことは未定．


## 発展
・ 最終的にスマホからブラウザで入力を行えるようにしたい．
